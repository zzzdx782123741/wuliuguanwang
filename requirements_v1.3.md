## 三、产品方案（V1.3 更新）

### 3.1 落地页方案（视觉交互升级）
*   **双入口Tab设计**：
    *   **位置**：顶部导航栏正下方居中悬浮（PC端）。
    *   **样式**：大尺寸胶囊型切换器，高亮选中态。
        *   **我要找车（货主）**：选中时显示 **万联红（Wanlian Red）** 渐变背景，配红色阴影。
        *   **我要找货（司机）**：选中时显示 **活力黄（Vitality Yellow）** 渐变背景，配黄色阴影。
    *   **交互**：点击即时切换内容，带有平滑过渡动画，页面自动滚动至顶部。

### 3.2 官网模块规划与内容呈现（视觉升级）

#### Tab A：【我要找车】（默认首页）
*   **整体风格**：商务、专业、高效。主色调为深蓝/灰搭配万联红。
*   **模块视觉优化**：
    1.  **Banner区**：全屏深色背景图 + 动态红蓝渐变遮罩 + 磨砂玻璃质感标签。
    2.  **核心痛点**：白色卡片悬浮效果 + 底部微透彩色背景装饰 + 动态Hover上浮交互。
    3.  **信任背书**：深色底纹 + 叠加企业Logo墙 + 关键数据高亮展示。
    4.  **数字化演示**：不规则流体渐变背景 + 3D透视效果的系统界面截图容器。
    5.  **留资转化**：全屏沉浸式背景图 + 红色半透明遮罩 + 玻璃态数据看板 + 悬浮表单。

#### Tab B：【我要找货】（司机端）
*   **整体风格**：活力、亲民、财富感。主色调为深黑/金黄搭配。
*   **模块视觉优化**：
    1.  **Banner区**：黑金风格背景 + 3D悬浮手机模型 + 醒目的红包福利入口。
    2.  **利益诱导**：大尺寸卡片布局 + 顶部彩色装饰条 + 线性渐变图标。
    3.  **信任数据**：深色星空背景 + 金色渐变数字大屏展示。
    4.  **底部下载**：大圆角容器 + 左右分栏布局 + 场景化背景图纹。
