# 整理与教学计划

## 1. 清理冗余文件
- **删除旧文件夹**：我们已经把 `guanwang-prototype` 里的代码都移到了最外层，这个旧文件夹现在是多余的了。刚才我尝试删除它失败了（可能是因为它不想离开主人），我会用强力一点的魔法（命令行）把它清理掉，让项目结构更干净。

## 2. 编写同步指南
- **创建教程文件**：我会为您专门写一个 `Git同步指南.md` 文件，放在项目里。
- **内容包含**：
    - 如何提交新的修改（三部曲：add, commit, push）。
    - 遇到冲突怎么办。
    - 简单的命令速查表。
